# Site and Content Navigation

This folder manages navigation structures and visual assets used to help users move through Naftiko's web properties and content. It supports the design and maintenance of menus, link trees, and navigational imagery. Work here includes defining navigation entries, updating site maps, and coordinating navigation-related images.

## Folders

- **entries** — navigation link entries and menu definitions
- **images** — images used in navigation elements and wayfinding
